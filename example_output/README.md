# Пример вывода (example_output)

В этой папке — **примеры выходных файлов** для показа в репозитории (формат отчёта и датасета).

| Файл | Описание |
|------|----------|
| `report_sample.txt` | Пример итогового отчёта по A/B-тесту (фрагмент). |
| `dataset_sample.txt` | Фрагмент датасета в текстовом виде. |

Полные результаты при запуске сохраняются в **output/** (не выгружается в репозиторий).
