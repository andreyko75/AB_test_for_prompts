# Промпт А (контроль) — пример

Используется как базовая версия в A/B-тесте.

---

На основании описания товара LR создай пост для телеграмм канала — описание товара с продающим текстом, ничего не придумывай, используй только точные подтвержденные данные о влиянии товара и ингредиентов на здоровье, которые можно взять не только из отправленного мной текста, но и используя общедоступные знания.
